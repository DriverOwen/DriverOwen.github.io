解释器和运算

注意点

  1. Python 全面支持浮点数；混合类型运算数的运算会把整数转换为浮点数：
  2. 用 // 运算符执行 floor division 的结果是整数（忽略小数)
  3. 交互模式下，上次输出的表达式会赋给变量 _,

    最好把该变量当作只读类型。不要为它显式赋值，否则会创建一个同名独立局部变量，该变量会用它的魔法行为屏蔽内置变量。

  4. python 中 ‘’ “” 都可以表示字符串
  5. 如果不希望前置 \ 的字符转义成特殊字符，可以使用 原始字符串，在引号前添加 r 即可
  6. 字符串字面值可以实现跨行连续输入。实现方式是用三引号：”””…””” 或 ‘’’…’’’，字符串行尾会自动加上回车换行，如果不需要回车换行，在行尾添加 \ 即可。示例如下：
  7. 字符串可以用 + 合并（粘到一起），也可以用 * 重复
  8. 字符串索引还支持负数，用负数索引时，从右边开始计数 -1表示最左边
  9. 内置函数 len() 返回字符串的长度

列表

  1. ython 支持多种 复合 数据类型，可将不同值组合在一起。最常用的 列表 ，是用方括号标注，逗号分隔的一组值。列表 可以包含不同类型的元素，但一般情况下，各个元素的类型相同：
  • 常用方法
    • list.append() 在list 末尾增加一个元素
    • list.insert（n,’4’） 在指定位置添加元素，如果指定的下标不存在，那么就是在末 尾添加
    • list1.extend(list2) 合并两个list list2中仍有元素
    • print(list) 遍历列表
    • list.pop() 删最后一个元素
    • list.pop(n)指定下标，删除指定的元素，如果删除一个不存在的元素会报错
    • list.remove(xx) 删除list 里面的一个元素，有多个相同的元素，删除第一个
    • del list[n] 删除指定下标对应的元素
    • del list 删除整个列表， list删除后无法访问
    • list.reverse()将列表反转
    • list.sort()排序，默认升序
    • list.sort(reverse=True) 降序排列

      注：list 中有字符串，数字时不能排序，排序针对同类型

    • len(list)：列表元素个数
    • max(list)：返回列表元素最大值
    • min(list)：返回列表元素最小值
    • list(seq)：将元组转换为列表
